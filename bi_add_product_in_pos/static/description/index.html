Add Multiple Products in POS in Single Click Odoo App

Set Mass Product Available in POS Odoo Apps

Allow Mass Product Available in Point of Sales Odoo Apps

POS Set Mass Product Available Odoo app helps users to add multiple selected products in the point of sale with a single click. Users do not need to enable the "Available in POS" option manually for each product, users can select multiple products from the tree view and make them available at the point of sale with a single click.

Features

Available Single or Multiple Product in POS

Users have the option to add single or multiple selected products to point of sale.

Availble Product in POS with Single Click

Users can add multiple products to point of sale screen at once with just a single click.
Demo Request Video Tutorial

Users can see "Available in POS" option is disabled for the product.

Users can also see that on point of sale screen products are not available.

Add Multiple Products in POS

From the product tree view user can select multiple products at once and click the "Available in POS" menu under the "Action" menu.

Users also can add products from a specific product form view.

Users can see the "Available in POS" option enable for all selected products.

Users can also see that product is availble on point of sale screen.
